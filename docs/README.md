# 📚 Документы для RAG

Эта папка содержит примеры документации Lua для демонстрации RAG-поиска.

## Доступные документы

| Файл | Темы | Размер |
|------|------|--------|
| [lua_tables.md](lua_tables.md) | Таблицы, ipairs, pairs | 2 KB |
| [lua_functions.md](lua_functions.md) | Функции, замыкания | 1.5 KB |
| [lua_modules.md](lua_modules.md) | Модули, require | 1 KB |
| [lua_strings.md](lua_strings.md) | Строки, patterns | 1.5 KB |
| [lua_error_handling.md](lua_error_handling.md) | pcall, xpcall | 1 KB |

## Как использовать

1. Загрузите файл через веб-интерфейс или API
2. Запросите генерацию кода по теме
3. RAG автоматически найдёт релевантные фрагменты
